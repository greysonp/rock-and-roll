# Rock and Roll

A top-down 2D action game where you control Rock, who is connected to Roll via a rope. Use the rope to defeat enemies while avoiding direct contact!

## Play Now

[Play on GitHub Pages](https://greysonp.github.io/rock-and-roll/)

## How to Play

**Objective:** Hit enemies with the rope between Rock and Roll without letting Rock or Roll touch any enemies!

### Controls

**Desktop:**
- **WASD** - Move Rock (blue circle)
- **Arrow Keys ↑↓** or **Scroll Wheel** - Control Roll's rotation speed
- **Slider** - Fine-tune rotation speed

**Mobile:**
- **Tilt Controls** - Tilt your device to move Rock (tap "Enable Tilt Controls" button)
- **Slider** - Control Roll's rotation speed

### Gameplay

- **Rock** (blue circle) - The character you control directly
- **Roll** (red circle) - Orbits around Rock on roller skates
- **Yellow Rope** - Connects Rock and Roll, destroys enemies on contact
- **Purple Enemies** - Spawn from the edges and move toward the center

The rope will destroy enemies it touches, increasing your score. Survive as long as possible!

## Local Development

Simply open `index.html` in any modern web browser. No build process required!

## Technologies

- HTML5 Canvas
- Vanilla JavaScript
- DeviceOrientation API for mobile tilt controls
